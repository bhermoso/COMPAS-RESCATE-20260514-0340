## PAN-001

Panóptico textual provisional — PROVISIONAL_NO_VALIDADO

**Propósito del bloque.** Las líneas 1 a 2000 de COMPÁS definen la totalidad de la capa de presentación visual del sistema. Sin excepción, todo lo que el usuario ve tiene su base en este bloque. No existe lógica funcional en este rango salvo un único listener de error en l.27–41.

**Composición.** Un único archivo HTML con un bloque `<style>` de aproximadamente 1960 líneas CSS activas que cubre 30 entidades visuales (más 1 transversal responsive) distribuidas en 9 capas funcionales. El bloque no cierra en l.2000 — continúa en CAP-002.

**Sistema de control de estado.** 20 contratos visuales (CON-001 a CON-020) permiten a JavaScript controlar qué se muestra sin manipular el DOM directamente. El patrón dominante es `clase_base { display:none } clase_base.modificador { display:block }`. Se aplica en: contenedores de vista (CON-001), fases RELAS (CON-002), acordeones (CON-003), determinantes (CON-007), árbol EPVSA (CON-008, CON-009), panel de carga (CON-012), agendas listado (CON-014), menú contextual (CON-017), toast (CON-019).

**Dependencias externas.** 6 CDN en l.13–25. Solo mammoth.js tiene evidencia de uso confirmada en este bloque (comentario l.808). Las demás 5 dependencias tienen evidencia probable o pendiente de validación. QRCodeJS es la única sin relación CSS observable.

**Riesgo sistémico mayor.** El color de fondo de al menos 6 componentes (.propuesta-linea-header, .modal-detalle-header, .modal-header, .agenda-tipo-linea-codigo, .badge-linea, .doc-linea) depende de JavaScript aplicando inline styles. Si JS no funciona o el dato de color está ausente, estos componentes muestran texto blanco sobre fondo blanco — invisible para el usuario. No detectable desde CSS solo.

**Deuda cromática.** 11 deudas documentadas. La más sistémica: `linear-gradient(X, color 0%, mismo_color 100%)` — degradado de un único color sin efecto visual — aparece al menos 6 veces (DV-007). La más impactante: uso de paleta Tailwind ajena al sistema (#ea580c, #f97316, #1e40af, #b91c1c, #047857) en componentes de alta visibilidad.

**Sistema de notificación.** ENT-029 (toast, z-index:10000) es el único mecanismo de feedback de usuario de alcance global en este bloque. Cualquier operación asíncrona (carga CSV, Firebase, generación de propuesta) depende de este componente para dar feedback visible. No existe overlay de carga a pantalla completa en este bloque.

**Complejidad de media queries.** 5 bloques @media dispersos en MR-029, 036, 042, 043, 044 para el breakpoint 768px. Sin punto de definición centralizado. El breakpoint 600px aparece en MR-040 y MR-044. El breakpoint 1200px aparece solo en MR-042.

**Jerarquía de capas z-index.** Confirmada en 3 niveles: z-index:100 (menú contextual tarjeta Kanban) < z-index:1000 (overlay modal) < z-index:10000 (toast). No se han detectado conflictos dentro del rango auditado.
