# PROTOTIPO-REACT-SALA-DE-MANDO-R1

Prototipo aislado de Sala de Mando para navegar la arquitectura documental PAN sin integrar COMPAS.

## Reglas visibles

- Solo lectura.
- No integrada en COMPAS.
- No modifica index.html.
- No gobierna runtime.
- No declara Control Total.

## Comandos

```bash
npm install
npm run dev
```

## Alcance R1

Incluye vistas ejecutiva, arbol, nodo, relaciones, riesgos, runtime, cadenas funcionales y busqueda global con datos estaticos locales.

No incluye Firebase, motores, dashboard funcional, persistencia real, autenticacion real ni integracion con el monolito.
